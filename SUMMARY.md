```

標題:
├─ H1: 2.5rem (行動) → 3rem (桌面)
├─ H2: 1.5rem → 2rem
├─ H3: 1.125rem → 1.25rem

文字:
├─ 正文: 1rem
├─ 小型: 0.85rem
└─ 標籤: 0.9rem

├─ 外邊距: 1rem - 2.5rem
```

## 💡 快速開始

### 本地測試
```bash
# 進入 frontend 目錄
cd frontend

# 使用 Python 3 啟動伺服器
python -m http.server 8000

# 訪問應用
# http://localhost:8000
```

### 測試頁面
- 首頁: http://localhost:8000
- 儀表板: http://localhost:8000/pages/dashboard.html
- 歷史: http://localhost:8000/pages/history.html
- 設定: http://localhost:8000/pages/settings.html

### 檢查響應式
按 F12 → Ctrl+Shift+M 切換到響應式設計模式
